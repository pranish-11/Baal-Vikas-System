🎓 BaalVikas

BaalVikas is a cloud-based Montessori Management System designed to enhance communication, transparency, and operational efficiency within educational institutions. The platform streamlines attendance management, behavior tracking, parent-teacher communication, complaint resolution, and daily activity monitoring through dedicated dashboards for administrators, teachers, and parents. With real-time updates, AI-powered assistance, and centralized record management, BaalVikas helps schools reduce administrative workload while keeping parents actively engaged in their child's educational journey.

## 🚀 Tech Stack

### 🎨 Frontend
- React 19
- Vite
- React Router 7
- Lucide React
- Socket.IO Client

### ⚙️ Backend
- Node.js
- Express.js
- Socket.IO

### 🗄️ Database
- MongoDB
- Prisma ORM
